# サウンドファイル

このディレクトリにはタイマー完了時の通知音を配置します。

## 必要なファイル

以下のMP3ファイルを追加してください：

- `default.mp3` - デフォルトのアラーム音
- `bell.mp3` - ベル音（オプション）
- `chime.mp3` - チャイム音（オプション）
- `beep.mp3` - ビープ音（オプション）

## サウンドファイルの入手方法

1. **無料サウンド素材サイト**:
   - Freesound.org
   - Pixabay
   - YouTube Audio Library

2. **自作する場合**:
   - Audacity などのオーディオ編集ソフトで作成
   - Web Audio API で生成（useSoundで対応予定）

3. **推奨仕様**:
   - フォーマット: MP3
   - ビットレート: 128kbps
   - 長さ: 1-3秒
   - サイズ: 50KB以下

## 注意事項

- ファイルサイズは小さく保つこと（PWAのキャッシュ効率のため）
- 著作権フリーの音源を使用すること
- 音量は適度に調整すること

## Web Audio API による代替実装

サウンドファイルがない場合、Web Audio APIを使用して
プログラムでビープ音を生成することも可能です。
